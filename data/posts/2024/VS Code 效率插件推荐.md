故事背景

作为前端开发者，VS Code 是我的主力工具。我整理了一些提升编码效率的插件，分享给大家。

设计思路

- 选择支持代码格式化、智能补全和调试的插件。
- 确保兼容性，避免冲突。
- 插件列表

- **Prettier**: 代码格式化。
- **ESLint**: 静态代码检查。
- **GitLens**: Git 集成。
- **Live Server**: 实时预览。
- 配置示例

json

```json
// .vscode/settings.json
{
  "editor.formatOnSave": true,
  "eslint.autoFixOnSave": true
}
```
